## 🎯 使用方式

### 1. 添加依赖

```xml
<dependency>
    <groupId>com.ron</groupId>
    <artifactId>javainfohunter-ai-service</artifactId>
</dependency>
```

### 2. 创建自定义 Agent

```java
@Component
public class MyAgent extends ToolCallAgent {

    @Autowired
    private ChatClient chatClient;

    public MyAgent() {
        super(toolRegistry.getAllTools());
        setName("MyAgent");
        setSystemPrompt("你是一个专家...");
    }
}
```

### 3. 使用 Agent 编排

```java
@Autowired
private AgentService agentService;

// Parallel 模式
CoordinationResult result = agentService.executeParallel(
    "任务描述",
    List.of("agent1", "agent2", "agent3")
);
```
